# Exact files to create/update
- `src/main/resources/css/app.css`

# Exact classes/interfaces
- CSS selectors used by JavaFX nodes and style classes.

# Exact method signatures
- Not applicable.

- Define primary/secondary button styles.
- Define panel spacing/margins.
- Define table row and header styling.

# Dependencies from other parts
- Applied by `MainApp` bootstrap.

# Out of scope
- Feature logic.
- Dynamic theming engine.

# Acceptance checklist
- App loads CSS with no warnings.
- Buttons/tables are visually consistent.
- UI remains readable in presentation/demo.
